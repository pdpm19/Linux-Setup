# Linux Setup

## Bash
```bash
sudo apt update
sh install.sh
```

## ZShell
```bash
sudo apt update
sh zsh.sh
```

## I3 keybinding
Run after setting up i3 for the first time
```bash
sh i3conf.sh
```

## JupyterLab
Open `Jupyter.md` file for basic configuration
